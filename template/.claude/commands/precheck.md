# /precheck

プッシュ前のチェックを実行するコマンド。

## 実行内容

1. 型チェック（該当する場合）
2. Lint
3. テスト実行
4. ビルド（任意）

## 手順

```bash
{{CHECK_COMMAND}}
```

## ビルドの確認

```bash
{{BUILD_COMMAND}}
```

## エラーが出た場合

### 型エラー
- エラーメッセージの行番号を確認
- 該当ファイルを修正

### Lint エラー
- 自動修正可能なものは lint の fix オプションで修正
- 手動修正が必要なものはエラーメッセージを確認

### テスト失敗
- 失敗したテストの詳細を確認
- ウォッチモードで実行しながら修正
